# online-store
1. Релизовать кастомный чекбокс на фильтре товаров;
2. Релизовать открытие и закрытие корзины в модальном окне;
3.  Релизовать корзину товаров:
	- добавление и удаление товаров в корзине;
	- счётчик количества товаров в корзине;
	- вывести общую сумму товаров в корзине.
4. Реализовать фильтр товаров в сайдбаре:
	- по акциям;
	- по цене.
	- **Важно:** если стоит галочка Акции, то при фильтре цены должны выводиться товары с акцией с этой ценой. Если уже сработал фильтр по цене, то поставив галочку акции, будут показываться товары по акции в этом ценовом диапазоне. При фильтре товаров в каталоге, фильтры цена и акция должны корректно работать, то есть выводить только те товары, категория которых была выбрана
5. Реализовать посик товаров в каталоге ;
6. Реализовать получение данных (директория db) и вывод товаров на страницу;
7. Реализовать каталог и написать его функционал (отображать категорию нужных товаров, котрая была выбрана). 
	- **Важно:** при фильтре товаров в каталоге, фильтры цена и акция должны корректно работать, то есть выводить только те товары, категория которых была выбрана.
